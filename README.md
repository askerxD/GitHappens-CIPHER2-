# GitHappens-CIPHER2-

The hackathon project for the CIPHER 2.0.

## Overview

Veritas is a library management interface built as a static HTML/CSS/JavaScript demo. It supports role-based dashboard navigation for students, staff, and admins, with reservation booking, availability browsing, QR check-in/out, notifications, issue reporting, and analytics.

## Setup

1. Clone or download the repository.
2. Open the project folder in your editor or file explorer.
3. No build step is required — this is a static website.
4. Open `index.html` in a web browser.

> Tip: For best results, use a modern browser such as Chrome, Edge, or Firefox.

## Usage

- On the login page, choose a role: Student, Staff, or Admin.
- Enter any University ID and password, then click **Sign in**.
- Navigate the dashboard using the sidebar and quick action buttons.
- Use the Student view to browse resources, make reservations, and view notifications.
- Use the Staff view to manage overdue items, view check-in activity, and handle override requests.
- Use the Admin view to review analytics, manage fair usage, and adjust priority settings.

## Files

- `index.html` — main application markup and page structure
- `style.css` — styling for layout, theme, and components
- `script.js` — application behavior, dashboard interactions, and demo data
